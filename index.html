BinSight is a smart waste-management system designed to reduce food waste using automation and real-time monitoring. It uses sensors, RFID technology, and microcontrollers to identify users, monitor waste levels, and take actions automatically.

Smart automated dustbin in action

ðŸ”¹ How does BinSight work?

When a user scans their RFID card, BinSight verifies the user and opens the dustbin lid automatically. After waste is thrown, ultrasonic sensors check whether the waste exceeds the allowed limit. If excess waste is detected, the system alerts through a buzzer and sends a notification. If the waste is within limits, it is safely disposed into the correct container.
